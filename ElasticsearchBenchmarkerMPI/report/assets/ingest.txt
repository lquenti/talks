usage: ingestor.py [-h] --data DATA --index INDEX --hosts HOSTS
                   [--bulksize BULKSIZE] [--indexname INDEXNAME]
                   [--shards SHARDS]

Load ingestor for Elasticsearch.

options:
  -h, --help            show this help message and exit
  --data DATA           Path to the JSON data to be ingested
  --index INDEX         Path to the Metadata Description to be ingested
  --hosts HOSTS         Comma separated hosts. Example: "--hosts
                        host1:9200,10.0.0.2:9200,localhost:9200"
  --bulksize BULKSIZE   Number of documents sent per request (defaults to
                        1000) (Optional)
  --indexname INDEXNAME
                        Name of the elasticsearch index created for the
                        specified data.
  --shards SHARDS       Number of shards, defaults to 1 shard per host
